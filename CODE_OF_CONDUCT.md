# Code of Conduct

This project follows a simple principle: **be respectful and constructive.**

- Assume good faith. People come from different backgrounds and skill levels.
- Critique ideas and code, not people.
- Keep it on-topic and kind. No harassment, discrimination, or personal attacks.
- Remember this is volunteer-maintained spare-time work; patience goes a long way.

Maintainers may edit, remove, or lock contributions that don't follow this, and
may block repeat offenders. If you see a problem, contact the maintainer via the
repo's contact options.

This is intentionally lightweight; for a fuller framework see the
[Contributor Covenant](https://www.contributor-covenant.org/).
